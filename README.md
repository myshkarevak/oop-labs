# OOP-labs
## Prerequisites
- minGW & g++
## How to use
1. Clone the repo
```
git clone https://github.com/myshkarevak/oop-labs
```
2. Change the directory
```
cd oop-labs
```
4. Choose the name of the lab, e.g. `lab1`
5. Run the start command using that name

For Powershell
```
.\run lab1
```
For cmd:
```
run lab1
```
